# create-scratch-extension <a href="https://npmjs.com/package/create-scratch-extension"><img src="https://img.shields.io/npm/v/create-scratch-extension" alt="npm package"></a>

A command-line application for creating scratch extensions.

## Creating Your First Scratch Extension

> **Compatibility Note:**
> This requires [Node.js](https://nodejs.org/en/) version 20.19+, 22.12+.

With NPM:

```bash
npm create scratch-extension@latest
```

With Yarn:

```bash
yarn create scratch-extension
```

With PNPM:

```bash
pnpm create scratch-extension
```

With Bun:

```bash
bun create scratch-extension
```

With Deno:

```bash
deno init --npm scratch-extension
```

## Templates

- `turbowarp-js`
- `turbowarp-ts`
- `penguinmod-js`
- `penguinmod-ts`
